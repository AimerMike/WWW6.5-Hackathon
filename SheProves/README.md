# SheProves · 她证

> 用她的方式理解数学，用她的语言教会彼此，用她的能力赚到第一笔钱。

---

## 一句话

高中女生把教科书的数学"翻译"成人话，同伴投票"我听懂了"，讲得好的人赚 AVAX。

---

## 问题

高中数学是女生放弃理科的第一道坎。但问题不是学不会——中国高考理科平均分男女差距不到 3%。问题是：

- 教材只有一种讲法，没听懂就以为自己不行
- 补课 ¥200/小时，讲法和课堂一样
- 会讲数学的女生没有任何渠道用这个能力赚钱
- "女生理科不行"这句话没有数据可以反驳

---

## 怎么做

### 核心动作：翻译

不是做题，不是问答。是把教科书语言**翻译**成自己的话。

```
┌─────────────────────────────────────────┐
│ 📖 教材原文                              │
│ "函数 f(x) 在 x₀ 处可导，当且仅当       │
│  极限 lim[f(x₀+Δx)-f(x₀)]/Δx 存在"     │
│                                           │
│ ────── ✨ 她的翻译 ──────                 │
│                                           │
│ "你在山坡上骑车，导数就是脚下这一小步      │
│  有多陡。Δx 是你迈的步子，越小越准。"     │
│                                           │
│ ✅ 312 人听懂 · 💰 已赚 0.8 AVAX          │
└─────────────────────────────────────────┘
```

同一个知识点可以有 50 种翻译。每一种都是原创内容，不是标准答案。

### 怎么赚钱

**悬赏**：不懂的人贴一段教材原文 + 附 AVAX → 多人提交翻译 → 社区投票 → 最佳者拿 90%，平台拿 10%。

**打赏**：好的翻译随手赏。95% 归作者，5% 归平台。

**品牌悬赏**：教培机构出资发起翻译挑战，获人才发现 + 品牌曝光。

### $PROVE：声誉代币

不是钱，是教学信用。翻译被验证 → 赚 $PROVE → 声誉高者排名靠前、被品牌签约、链上可查用于升学。

---

## 为什么上链

1. **悬赏资金锁在合约里**，不经过平台账户，自动分账
2. **"312 人听懂了"是链上共识**，不是数据库数字，任何人可验证
3. **STEMIndex 合约实时公开**：多少女生在贡献、多少翻译被验证——"女生能教数学"变成链上事实

---

## 为什么是 Avalanche

每次"听懂了"投票都是一笔链上交易。Avalanche gas < $0.01，亚秒确认，高频互动才经济可行。

**Subnet 愿景**：未来创建教育专用链——验证者不是质押资金的人，而是教学声誉最高的女生；gas 用 $PROVE 支付，通过教学赚取。**教数学的能力就是算力。**

## 技术

| 合约 | 功能 |

| `ExplanationManager.sol` | 翻译提交、投票、悬赏（创建/回答/分账/超时结算）、打赏 |

| `SheProvesCredential.sol` | Soulbound 成就凭证 |
